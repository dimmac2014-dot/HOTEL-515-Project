# 🏨 Hotel Quality Intelligence — Portfolio Project

> An end-to-end data analysis project simulating the work of a **Hotel Quality Coordinator**, using 515K guest reviews from Booking.com across Europe.

---

## 📌 Project Overview

This project analyzes hotel guest reviews to extract actionable quality insights — replicating real workflows used by Quality Coordinators in the hospitality industry.

**Dataset:** [515K Hotel Reviews Data in Europe](https://www.kaggle.com/datasets/jiashenliu/515k-hotel-reviews-data-in-europe) (Kaggle)

## 🔬 Analysis Phases

### Phase 1 — EDA
- Score distribution & statistics
- Top/Bottom hotels by country
- Seasonality (monthly trends)
- Reviewer nationality analysis

### Phase 2 — Complaint Categorization
- Keyword-based classification into 8 departments
- Cleanliness · Service · Food & Bev · Noise · Room · Location · Value · Facilities
- Complaint heatmap by country
- Impact of each complaint category on score

### Phase 3 — Sentiment Analysis
- VADER compound scoring on positive/negative reviews
- Correlation between text sentiment and numerical rating
- Mismatch detection (positive text + low score)
- Word clouds for positive vs negative reviews

### Phase 4 — Guest Segmentation
- Analysis by traveler type: Solo · Couple · Family · Business · Group
- Top complaint keywords per segment
- Business vs Leisure deep dive
- Length of stay vs satisfaction

### Phase 5 — KPI Dashboard
- NPS simulation (Promoters / Passives / Detractors)
- Interactive score trend over time
- Complaint department breakdown
- Europe choropleth map
- Hotel leaderboard
